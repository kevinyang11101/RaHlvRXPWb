# 汽车服务管理系统 java204

## 项目概述

汽车服务管理系统是一个多功能的汽车行业解决方案，集成了协同过滤推荐算法，通过Spring Boot技术实现。系统旨在提供一站式的汽车销售、配件购买、资讯浏览以及用户和员工管理。

## 技术栈

- **后端技术**：
  - Spring Boot
  - MyBatis
- **前端技术**：
  - Vue
  - ElementUI
  - Axios
  - EChart
  - HTML
  - JavaScript
  - CSS

## 系统角色

系统定义了以下两种角色：

- 管理员
- 用户

## 运行环境

- 开发工具：IDEA
- JDK版本：1.8
- 数据库：MySQL
- 依赖管理：Maven
